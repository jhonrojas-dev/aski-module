Ask your ERP in natural language

Connect your Odoo to Aski in one step — scan a QR, no typing.

Aski lets you ask your Odoo questions in plain language and get real figures in seconds — sales, receivables, top products, inactive customers and more — straight from your phone, even by voice.

This lightweight connector removes the manual setup. Install it, click Connect with Aski, and a one-time QR code is generated. Open the Aski app, scan it, and your phone is securely linked to this Odoo — no need to type URLs, databases or API keys by hand.

See it in action

Aski running on your phone — chat & voice, real figures from your Odoo.

How it works

  1. Install this module and open the new Aski menu.
  2. Click Generate connection code — a QR appears.
  3. Open the Aski mobile app, choose Scan code, point at the QR.
  4. Done. Start asking your ERP in plain language.

What you can ask

Aski is an AI assistant and chatbot for Odoo: ask your ERP in natural language and get instant answers and mobile reports — sales, receivables, top products, inactive customers, cash flow — by chat or by voice. A simpler alternative to building dashboards or BI reports for everyday questions.

Keywords: AI, assistant, chatbot, natural language, ask Odoo, mobile reports, business intelligence, BI, dashboards, voice, analytics, conversational.

Safe by design

The module only shows you a connection code. The Aski app connects directly to your Odoo through the standard external API. The code uses a standard Odoo API key generated for your user, which you can revoke anytime in Settings > Users > Account Security > API Keys.

Works with Odoo Community and Enterprise, versions 14 to 19.

Aski te permite preguntarle a tu Odoo en lenguaje natural y obtener cifras reales en segundos — ventas, cuentas por cobrar, productos más vendidos, clientes inactivos y más — desde tu celular, incluso por voz.

Este conector ligero elimina la configuración manual. Instálalo, haz clic en Conectar con Aski y se genera un código QR de un solo uso. Abre la app de Aski, escanéalo, y tu celular queda enlazado de forma segura a este Odoo — sin teclear URLs, bases de datos ni API keys.

Cómo funciona

  1. Instala este módulo y abre el nuevo menú Aski.
  2. Haz clic en Generar código de conexión — aparece un QR.
  3. Abre la app móvil de Aski, elige Escanear código y apunta al QR.
  4. Listo. Empieza a preguntarle a tu ERP en lenguaje natural.

Qué puedes preguntar

Aski es un asistente de IA y chatbot para Odoo: pregúntale a tu ERP en lenguaje natural y obtén respuestas instantáneas y reportes móviles — ventas, cuentas por cobrar, productos más vendidos, clientes inactivos, flujo de caja — por chat o por voz. Una alternativa más simple a armar dashboards o reportes de BI para las preguntas del día a día.

Seguro por diseño

El módulo solo te muestra un código de conexión. La app de Aski se conecta directamente a tu Odoo mediante la API externa estándar. El código usa una API key estándar de Odoo generada para tu usuario, que puedes revocar cuando quieras en Ajustes > Usuarios > Seguridad de la cuenta > API Keys.

Compatible con Odoo Community y Enterprise, versiones 14 a 19.
